Неделя целиком

Этот файл можно разместить на GitHub Pages: положите рядом index.html и recipes.json в один репозиторий, включите Pages в настройках репозитория — и дашборд будет открываться с телефона по ссылке. План, цели и свои рецепты хранятся в памяти браузера на устройстве (localStorage), библиотека книги — в recipes.json рядом с файлом. Обновили recipes.json в репозитории — нажали «Обновить библиотеку» на вкладке «Библиотека рецептов» — изменения подтянулись без переустановки.
